# Refugee Integration and Population Change in Germany (2008–2023)

This project analyzes the impact of the 2015 migration crisis on Germany’s population and refugee integration. It combines macro-level data on refugee inflows with population statistics to examine demographic trends over time.

---

## Objectives

- Examine refugee trends before and after 2015  
- Analyze population changes in Germany  
- Evaluate the relative impact of refugees on total population  

---

## Data Sources

- UNHCR – refugee and asylum seeker statistics  
- World Bank – total population of Germany  
- IZA Institute of Labor Economics – refugee integration dataset (documentation)  

---

## Methodology

- Time series analysis (2008–2023)  
- Data cleaning and transformation  
- Dataset merging using the year variable  
- Calculation of refugee share of population  
- Visualization of trends and distributions  

*Note: Micro-level data is available only after 2015.*

---

## Requirements

This project is developed in Python and executed in a Jupyter Notebook environment. The following libraries are required:

- pandas (data manipulation and processing)  
- matplotlib (data visualization)  

The project can be run using Anaconda or any Python environment with the required libraries installed.

---

## Results

The analysis shows a significant increase in refugee numbers during the 2015 migration crisis, followed by a period of stabilization. While absolute numbers increased sharply, the share of refugees relative to the total population remained moderate.
